## 🚀 QUICK START GUIDE

### 1. AUTOMATIC SETUP (Recommended)

Windows:
  > setup.bat

Linux/Mac:
  $ bash setup.sh

### 2. RUN DEMO (No Hardware Required)

  $ python main.py --mode demo

Expected output: Complete signal processing, ML, and clinical demonstration

### 3. RUN GUI (With Hardware)

Auto-detect:
  $ python main.py --mode gui

Specify port:
  $ python main.py --mode gui --device COM3

### 4. RUN CLI (Command-Line)

  $ python main.py --mode cli --device COM3

### 5. RUN TESTS

  $ python -m pytest tests/test_all.py -v
